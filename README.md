# Cloud-Based Smart File Storage & Sharing System

## Project Overview
This project is a cloud-based file storage and sharing system, similar to Google Drive, that allows users to upload, download, and share files securely through a web interface. The system is designed for scalability and ease of use, leveraging cloud services for storage and hosting.

## Features
- User registration and login
- Upload and download files
- Generate shareable links
- Secure and scalable cloud storage

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Cloud Storage:** AWS S3
- **Database:** MongoDB (Atlas or local)
- **Deployment:** AWS EC2 (Backend), AWS S3 + CloudFront (Frontend)
